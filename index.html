Whitepaper

Decentralized computing platform that targets low-end devices.

Near is a highly scalable base protocol that supports dApps fast enough to bridge the gap to seamless mainstream usage on mobile devices

Whitepaper

  • Scalable

    The network’s power scales as its size grows so it supports large commercial-grade dApps.

  • Fast

    Extremely fast finality times allow rapid peer-to-peer transactions.

  • Mobile

    Lite nodes run natively on mobile devices and give end users a simple gateway to the dApp ecosystem.
